

# Finance Education Website with AI Assistant

## Overview
A professional financial education platform where users can learn about personal finance, investing, and money management through curated content. An AI-powered chatbot will help users navigate the site, understand financial concepts, and answer their questions.

---

## Core Pages & Features

### 1. Landing Page
- Hero section with compelling headline about financial literacy
- Key benefits of the platform (learn, grow, succeed)
- Featured content highlights
- Call-to-action to sign up or start learning
- Clean, trustworthy corporate design with professional typography

### 2. Educational Content Hub
- **Topic Categories**: Investing, Budgeting, Taxes, Retirement, Credit & Debt
- **Article Cards**: Thumbnail, title, reading time, difficulty level
- **Content Organization**: Browse by category or view all articles
- Search functionality to find specific topics

### 3. Article/Lesson Pages
- Clean, readable article layout
- Key takeaways highlighted
- Related articles suggestions
- Bookmark functionality for logged-in users

### 4. AI Chatbot Assistant
- Floating chat button accessible on all pages
- Expandable chat window with conversation history
- **Capabilities**:
  - Answer questions about financial concepts
  - Help users navigate the site
  - Explain terms like compound interest, ETFs, credit scores
  - Suggest relevant content based on questions
- Powered by Lovable AI (Gemini) for intelligent, grounded responses

### 5. User Dashboard
- Welcome message with personalized greeting
- Bookmarked/saved articles
- Recently viewed content
- Quick access to continue learning

### 6. Authentication System
- Sign up with email/password
- Login page
- Password reset functionality
- User profile management

---

## Design Direction
- **Color Palette**: Navy blue, white, subtle gold accents (trust & professionalism)
- **Typography**: Clean, modern sans-serif fonts
- **Layout**: Generous whitespace, card-based content, clear hierarchy
- **Overall Feel**: Trustworthy like a bank, but approachable and modern

---

## Technical Implementation
- **Frontend**: React with Tailwind CSS for professional styling
- **Backend**: Lovable Cloud for database and authentication
- **AI Chatbot**: Lovable AI integration via edge function for real-time responses
- **Data Storage**: User accounts, bookmarks, and reading history
